18 // DfgVertex which represents a storage location (e.g.: DfgVarPacked), or has multiple sinks
19 // without driving a storage location (and hence needs a temporary variable to duplication). The
20 // recursion stops when we reach a DfgVertex representing a storage location (e.g.: DfgVarPacked),
21 // or a vertex that that has multiple sinks (as these nodes will have a [potentially new temporary]
22 // corresponding// storage location). Redundant variables (those whose source vertex drives
23 // multiple variables) are eliminated when possible. Vertices driving multiple variables are
24 // rendered once, driving an arbitrarily (but deterministically) chosen canonical variable, and the
25 // corresponding redundant variables are assigned from the canonical variable.
